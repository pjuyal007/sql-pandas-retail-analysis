# 📊 SQL + Pandas Retail Analysis (Notebook)

---

## 🎯 Objective

To perform end-to-end data analysis by extracting data using SQL and analyzing it using Pandas to generate meaningful business insights.

---

## 📌 Overview

This notebook demonstrates how SQL and Pandas can be combined to build a powerful data analysis workflow. Data is fetched from a MySQL database using SQL queries and further processed using Pandas for deeper analysis.

---

## 🔄 Workflow

* Connected to MySQL database using SQLAlchemy
* Executed SQL queries directly from Python
* Loaded query results into Pandas DataFrames
* Performed data transformation and feature engineering
* Generated insights and exported data for dashboard creation
